Release v0.3.0 is out! This update brings comprehensive end-to-end flows for storyboard to export, a production-grade PDF export path, and a hardened accessibility/test suite.

- End-to-End Flows A-F (CSV, PDF, Pack exports)
- Production PDF export using pdf-lib
- CSV export and JSON Pack export
- Accessibility and playback-rate hardening
- GitHub Actions CI for unit and end-to-end tests

How to try it locally:
- npm install
- npm run dev
- npm run test or npm run test:e2e
